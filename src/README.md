# Source Code

Este directorio contiene los scripts y notebooks que constituyen el núcleo del proyecto de análisis de sentimiento para las reseñas de productos deportivos de Amazon.

## Estructura del Directorio

- `1_descarga_datos.ipynb`: Notebook que maneja la descarga y la carga inicial de los datos.
- `2_preprocesado.ipynb`: Procedimientos de preprocesamiento de los datos antes del análisis.
- `3_entrenamiento_y_testeo_modelo.ipynb`: Notebook dedicado al entrenamiento y evaluación de los modelos de machine learning y deep learning.
- `preprocessing.py`: Script que contiene funciones de preprocesamiento de texto reutilizables.
- `utils.py`: Utilidades generales utilizadas a lo largo del proyecto, como funciones de carga y guardado de modelos y funciones para visualizar los datos.
